# README.1st

## Descripción del Proyecto

Este proyecto (`ind_inv_raiz.py`) está diseñado para procesar un archivo de URLs (`urls.txt`) y generar un índice invertido representado por un diccionario en un archivo de salida (`raiz_ind_inv.txt`). La raíz de las palabras se utiliza como clave, y el valor es una lista de tuplas (URL, frecuencia).

## Archivos del Proyecto

- **ind_inv_raiz.py:** Contiene el código principal para procesar las URLs y generar el índice invertido.
- **urls.txt:** Archivo de entrada que debe contener al menos diez mil URLs diferentes.
- **raiz_ind_inv.txt:** Archivo de salida que contiene el índice invertido generado.

## Ejecución del Programa

Asegúrate de tener Python instalado en tu máquina.

1. **Clona el Repositorio:**
    ```bash
    git clone <URL_del_repositorio>
    ```
   Reemplaza `<URL_del_repositorio>` con la URL de tu repositorio en GitHub.

2. **Navega al Directorio:**
    ```bash
    cd ind_inv_raiz
    ```

3. **Ejecuta el Programa:**
    ```bash
    python ind_inv_raiz.py
    ```

4. **Verifica el Resultado:**
   - El índice invertido se guardará en el archivo `raiz_ind_inv.txt`.
